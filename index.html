Locasberos

https://github.com/dgvncsz0f/locasberos

An umbrella for three projects:

* caslib: A CAS client library, written in ANSI C [1];
* apache-mod-locasberos: An apache module for CAS authentication using caslib;
* nginx-mod-locasberos: The same as above, but for nginx;

The project was made using C language and libcurl for HTTP requests.

Robotz Warehouse

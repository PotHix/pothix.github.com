Usando Git em projetos do Google Code

Published at May 11, 2009 tech,
Resolvi escrever esse post depois de começar a contribuir para o projeto do vimbook, que está no Google Code. Como o Google Code só usa Subversion (infelizmente) e eu sou do mundo Git, resolvi usar a ferramenta Git para lidar com SVN, o git-svn! Com ele você pode clonar um repositório SVN e utilizar como Git, e na hora de “commitar” ele retransforma em SVN. Já havia tentado usar Git-svn no meu projeto de plugin para wordpress, mas nesse caso eu já tinha um repositório Git e fui tentar mandar para um repositório SVN vazio, o que só me deu dor de cabeça, e no fim das contas acabei jogando para um repositório SVN qualquer só para mandar para o Wordpress.
